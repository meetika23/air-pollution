Linear Regression

Linear Regression is a supervised machine learning algorithm that is used for predictive analysis. The linear regression technique is used to make predictions on continuous or numeric variables.Linear regression algorithm shows a linear relationship between a dependent (Y) and one or more independent (X) variables, hence called as linear regression. Since linear regression shows the linear relationship, which means it finds how the value of the dependent variable is changing according to the value of the independent variable.


How it works?

This web application predicts the Air Quality Index (AQI) by taking in levels of four major air pollutants - Sulphur dioxide (SO2), Nitrogen dioxide (NO2), Respirable Suspended Particulate Matter (RSPM), and Suspended Particulate Matter (SPM). To make these predictions, it uses Linear Regression to analyze the pollutant levels and predict how they will impact air quality.
